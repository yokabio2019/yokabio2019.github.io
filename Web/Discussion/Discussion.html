    Mismatch

    Besides,the experiment for the ON/OFF switch mechanism of fluorescent molecules isconducted byusing mismatched base pairs. As a result, an ON / OFF mechanism is achieved, but it did not reachthe point where the convergence value decreased from the initial concentration or the time of the OFF state was controlled. The current switch circuitdelay mechanism using mismatched base pairs can be implemented with a seesaw gate and is expected to be achievedin future research. Also, as an alternative tomismatched base pairs,special self-bindings such as hairpin loops and bulge loops can be used to implement this idea.

    For display

    発展的に“画素”とするアイデアを提案したい。一般にカラーディスプレイに使われている画素はR(red),G(green),B(blue)の三原色を用いて表現されている。これに倣うようにDNAorigamiでつくった平面構造に三原色をそれぞれ発する蛍光分子をとりつける。三原色をもとに表現したい色を蛍光させるために、これら蛍光分子が対応する消光分子に向かって計算しながら近づけて反応させることで、様々な色を表現することが可能になると考える。ただ、ここで一枚の平面に三つの蛍光分子をつけることによってフレットが生じる可能性を考慮しなければならない。しかしながら、DNAは4つの塩基を用いており、DNA origamiでつくったプレートには設計図が存在する。ゆえに、フレットが起きない距離を計算して、そこに蛍光分子をそれぞれ付着させるdesignを可能にすることができると考える。

    ELSI

    今回のプロジェクトに用いたのはDNAoを含めほとんどの部分がDNAで構成されている。そのため生体との親和性が高く、通常では起こりえない化学反応が起きる可能性を秘めている。今回のプロジェクト及び実験では生体内での利用を想定していないが、もし第三者がこれを悪用しようとした場合、生体内に注入して犯罪の道具として使用できてしまう。また、DNA鎖が生体内に投入されてしまった場合生物内にもともと存在する遺伝子への影響も考えなければならない。
